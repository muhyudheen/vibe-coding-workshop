# Vibe Coding Workshop

A collection of interactive web demos and coding experiments.

## Demos

### Vibe Royale

A fun battle royale-style game featuring 100 emoji players battling it out as a shrinking storm circle eliminates anyone caught outside. The last emoji standing wins!

**Link:** [vibe-royale.html](./vibe-royale.html)

**Usage:** Open vibe-royale.html in your browser to play.

**Tip:** Press `R` to restart the demo at any time.
